Table linkage_group
This table will contain different linkage groups, ie. Chromosome 1, Chromosome 2, etc. along with their respective start and stop boundaries.
* linkage_group_id serial
* name varchar( 2147483647 ) ex. Chromosome 1, Chromosome 2, ..., ChromosomeN, LG01, LG02, etc.
* start integer DEFO 0 Start of the linkage group. 0-based, interbased coordinates.
* stop integer DEFO 0 The maximum position in the linkage group, ex. 200, 200000000
* map_id integer Foreign key to the Map table. This defines which map the linkage group belongs to.
Indexes
Pkpk_linkage_group ON linkage_group_id
 idx_linkage_group ON map_id
Foreign Keys
 fk_linkage_group_map ( map_id ) ref mapset (mapset_id)
